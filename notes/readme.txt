Usage Instructions and Program Structure

	Starting the Program
		
		After running the script, click "Update Labels" to show the labels and links between them.
		
	Editing label data
		
		Right click on any label to load it into the Note Info view. Alternatively, search for any name not displayed.
		Change the text in any field, except the "Name" field, and click "Save Data"
	
	Making a new label

		Enter a unique name into the "Name" field and search. If it does not exist, all fields will be cleared
		Add the relevant data to the fields
		Click the "Save Data" button, then "Update Labels"
